Git Command Notes: 


1. for status:- git status
2. For initiate :- git init
3. Add all  Files :- git add --a 
4. Commit with message:- git commit -m "Initial Commit" 
5. For git log :- git log 
6. Add Files:- git add name.ext
7. For delete git repository:- rm-rf .git
8. Present Working Directory:- pwd
9. List Content(folder files list) :- ls
10. Change Directory:- cd
11. Git colne :- git clone https://github.com/tensorflow/data-validation.git
12. Create New File :-  touch file.ext
13. create gitignore file:-  touch .gitignore
14. for ignore ext wise all files :- update gitignore file with :-   *.log , *.txt
15. For Ignore Directory add this into gitignore file: Directoryname/
16. Git File Diffrence(compare staging area to working directory):- git diff
17. git Add files :- git add .
18. Remove file from floder :- git rm file.ext
19. Rename file :- git mv file.ext file_rename.ext 
20. Dont Want track file anymore :-  git rm --cached file.ext 
21. log with diffrence :- git log -p 
22. log with last 3 changes :- git log -p -2 
23. Short Summery of git commits Log:- git log --stat
24. Show all commits in Oneline Formate:- git log --pretty=oneline
25. Show all commuits in short :- git log --pretty=short 
26. Show all commits with full details :- git log --pretty=full
27. time filter in git log :- git log --since=2.days
28. time filter in git log :- git log --since=2.months
29. time filter in git log :- git log --since=2.years
30. Change in old commit:- git commit --amend  (add or update escap then :wq+ enter)
